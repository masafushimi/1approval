PHARMA EVENT PAIN POINTS

講演会・医師手配の「システム分断」が、
重大なコンプライアンス＆工数ロスを招いています

MRによる企画、手配業者への都度依頼、医師規程の個別チェック、そして透明性ガイドラインへの勘定科目手動マッピング。すべて手入力で行う体制は限界を迎えています。

医師規程の「目視チェック」限界

医師の役職や社内規定（HCO/HCPランク）に応じた「座席クラス、宿泊上限規定」のチェックをMRが手作業で確認。規程外の手配が監査段階で発覚し、二度手間やコンプラ違反の原因に。

会場手配のスピード不足とタイムラグ

MRからメールや代理店経由で会場の見積もりを取得するまでに数日を浪費。医師（演者）のスケジュールが超多忙な中、確認中に他社に会場を押さえられる機会損失が多発。

透明性ガイドラインの仕訳地獄

講演会費、情報交換会費、一般茶菓・弁当代、医師宿泊費等、証憑を見ながら「交際費」「非交際費」の厳密な切り分け、自社EOSS/EBOSSコードを手動で入力・仕訳する月末処理が莫大な負担に。

INTEGRATED WORKFLOW

手配から購買、仕訳までシームレスな一気通貫フロー

MRはSalesforce/Veeva上のアクションだけで、すべてのノンコア業務を自動化します。

01
Veeva / SF 起案

講演会企画の起案

MRが使い慣れたVeeva Events Management等から日程、場所、人数を入力して企画を起案。

02
コンシェルジュ

半自動会場提案

入力された内容に基づき、専門コンシェルジュがチャットや専用ポータルで最適な仮キープ済みの会場を半自動で提案。

03
1-Click 同期

仮予約情報の同期

確定した仮予約会場（または事前に入力した持込会場）のスペック・料金データを1クリックでSalesforceに完全同期。

04
承認＆本予約

承認時に自動購買・発券

上長承認アクションと同時に、手配システムが自動で本予約確定。医師のホテルや新幹線割引・LCC運賃まで自動発券。

05
ERP 出力

透明性ガイドライン連携

講演会の終了後、手配した実績データを自動で仕訳分類し、ConcurやSAPなどの主要基幹会計システムに直接出力。

POST-VENUE OPTIONS

宿泊交通手配オプション

招請医師（演者・座長・一般参加者）のホテル宿泊、新幹線チケット、国内航空会社（LCC12社含む）を一括キープ。HCP規定の宿泊上限や座席グレード自動チェック機能と完全に連動し、旅費経費としてSalesforceに自動マッピング。

対応EOSS：0109 一般参加者旅費等 自動仕訳連携

お弁当手配オプション

講演会当日用のお弁当や、控室・情報交換会用のケータリング・茶菓の手配を行います。軽減税率（8%）と通常税率（10%）、非交際費（役員・参加者弁当代）の切り分けなどの複雑な財務仕訳情報を、手配完了と同時に自動でSalesforceの経費タイプへ反映。

対応EOSS：0110 一般参加者弁当代 軽減税率(8%)対応

備品手配オプション

イベントに必要な音響映像機材（マイク、スクリーン、プロジェクター、配信カメラ等）や装飾看板、翻訳・通訳者の手配、印刷物（テキスト・アジェンダ等）の手配を一括管理。コンシェルジュが空き状況確認から当日セッティングまでをサポートします。
